# 信贷知识库

## 法规文件

### 1. 个人信贷风险管理指引
**来源**: 银保监会规定
**内容**: 金融机构应建立完善的个人信贷风险评估体系，包括但不限于：
- 借款人还款能力评估
- 信用记录审查
- 负债情况调查
- 贷款用途核实

### 2. 贷款审批合规要求
**来源**: 内部合规手册
**内容**: 贷款审批应遵循审慎原则，确保借款人具备还款能力：
- 收入稳定性验证
- 负债率控制在合理范围
- 信用记录良好
- 贷款金额与收入匹配

### 3. 个人贷款管理办法
**来源**: 银保监会令
**内容**: 
- 贷款人应对借款人进行资信评估
- 贷款金额应根据借款人收入水平确定
- 应建立逾期贷款催收机制
- 超过一定金额需提供担保或抵押

### 4. 商业银行授信指引
**来源**: 中国人民银行
**内容**:
- 严格执行贷款三查制度（贷前调查、贷中审查、贷后检查）
- 加强贷款用途管理
- 建立风险预警机制
- 完善内部评级体系
